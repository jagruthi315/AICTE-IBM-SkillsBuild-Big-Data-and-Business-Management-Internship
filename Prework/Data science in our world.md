# What is Data Science?

Data Science is the process of extracting meaningful insights from data using:

- Scientific Method
- Mathematics & Statistics
- Programming
- Machine Learning
- Artificial Intelligence
- Data Visualization
- Storytelling

### Goal

Transform raw data into actionable business insights and make predictions for better decision-making.

---

# Data Analytics vs Data Science

| Data Analytics | Data Science |
|---------------|-------------|
| Works mostly with structured data | Works with structured & unstructured data |
| Cleans, analyzes and visualizes data | Builds predictive models |
| Finds trends and insights | Predicts future outcomes |
| Uses SQL, Excel, Power BI, Tableau | Uses Python, ML, AI, Statistics |

### Common Skills

- Analytical Thinking
- Critical Thinking
- Python
- Data Storytelling
- Problem Solving

---

# Curiosity - The Most Important Skill

A good data scientist constantly asks:

> **Why?**

The **5 Whys Technique** helps identify the root cause of a problem by repeatedly asking "Why?" until the actual cause is found.

Example:

Problem → Employee attrition is increasing

- Why?
- Why?
- Why?
- Why?
- Why?

Root Cause → Employees live too far from the office.

# Typical Data Science Workflow

## 1. Business Understanding

- Understand the business problem
- Define objectives
- Decide analytical approach

---

## 2. Data Exploration & Preparation

- Collect data
- Clean missing values
- Remove duplicates
- Prepare dataset

---

## 3. Data Representation & Transformation

Understand the dataset using:

- Mean
- Median
- Mode
- Standard Deviation
- Min & Max

Perform feature engineering and transform data into model-friendly format.

Example:
Convert **Yes/No** into **1/0**.

---

## 4. Data Visualization

Purpose:
Convert insights into easy-to-understand visuals.

Common Charts:

- Bar Chart
- Column Chart
- Line Chart
- Pie Chart
- Scatter Plot

Good visualizations should be:

- Clear
- Attractive
- Meaningful
- Easy to interpret

---

## 5. Train Data Models

Machine Learning allows computers to learn patterns from data.

### Types

### Supervised Learning

- Uses labeled data
- Predicts known outcomes

Examples:

- Spam Detection
- Fraud Detection
- House Price Prediction

---

### Unsupervised Learning

- Uses unlabeled data
- Finds hidden patterns

Examples:

- Customer Segmentation
- Recommendation Systems

---

### Reinforcement Learning

- Learns through rewards and penalties.

Examples:

- Robotics
- Self-driving cars
- Game AI

---

## 6. Deploy Models

After training:

- Save the model
- Deploy into production
- Monitor performance
- Retrain when required

# Key Takeaways

- Data Science combines statistics, programming, AI, and business understanding.
- Curiosity is one of the most important qualities of a data scientist.
- Data science follows a structured methodology rather than random experimentation.
- Machine Learning is used to build predictive models.
- Data visualization communicates insights effectively.
- Data Science is a collaborative effort involving Analysts, Scientists, and Engineers.
- The ultimate goal is to solve business problems using data.
