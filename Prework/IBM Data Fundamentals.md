# 📊 Data & Analytics Overview

## 1. Importance of Data
Data is one of the most valuable assets for organizations. Businesses use data to:
- [Improve operations](ca://s?q=How_data_improves_operations)
- [Understand customer behavior](ca://s?q=Understanding_customer_behavior_with_data)
- [Reduce costs](ca://s?q=How_data_reduces_costs)
- [Increase profits](ca://s?q=How_data_increases_profits)
- [Make informed decisions](ca://s?q=Data_driven_decision_making)
- [Drive innovation](ca://s?q=Data_driven_innovation)

➡️ Data analytics converts raw data into meaningful business insights.

---

## 2. Types of Data

### [Structured Data](ca://s?q=Structured_data_examples)
- Organized in rows and columns
- Easy to store and query
- Used in relational databases  
**Examples:** Customer IDs, Dates, Sales records  
**Tools:** SQL, Excel

### [Semi-Structured Data](ca://s?q=Semi_structured_data_examples)
- Partially organized
- Doesn’t follow a fixed table format  
**Examples:** JSON, XML, Emails

### [Unstructured Data](ca://s?q=Unstructured_data_examples)
- No predefined structure
- ~80% of the world’s data  
**Examples:** Images, Videos, PDFs, Audio, Social media posts

---

## 3. Database Basics
A [database](ca://s?q=Database_basics) is an organized collection of structured data.  
Most businesses use **Relational Databases**, where multiple tables are connected using keys.  

**Example:**  
Customers → Orders → Products  

➡️ SQL is used to retrieve and manipulate data.

---

## 4. Quantitative vs Qualitative Data

### [Quantitative Data](ca://s?q=Quantitative_data_examples)
- Numerical data  
**Examples:** Height, Weight, Sales, Temperature  

**Types:**
- **Discrete:** Countable values (e.g., Number of employees)  
- **Continuous:** Measurable values (e.g., Height, Weight)

### [Qualitative Data](ca://s?q=Qualitative_data_examples)
- Categorical data  
**Examples:** Gender, City, Color  

**Types:**
- **Nominal:** No order (e.g., Blood Group, Country)  
- **Ordinal:** Ordered categories (e.g., Poor, Good, Excellent)

---

## 5. Big Data
[Big Data](ca://s?q=Big_Data_5Vs) refers to datasets too large and complex for traditional processing methods.

**The 5 Vs:**
- Volume → Amount of data  
- Velocity → Speed of data generation  
- Variety → Different types of data  
- Veracity → Data quality  
- Value → Business usefulness

## 6. Types of Data Analytics

| **Type**        | **Answers**                |
|------------------|----------------------------|
| [Descriptive](ca://s?q=Descriptive_data_analytics) | What happened? |
| [Diagnostic](ca://s?q=Diagnostic_data_analytics)   | Why did it happen? |
| [Predictive](ca://s?q=Predictive_data_analytics)   | What is likely to happen? |
| [Prescriptive](ca://s?q=Prescriptive_data_analytics) | What should be done? |

## 7. Data Analysis Process
IBM introduced a simple workflow:

**Collect → Clean → Analyze → Visualize**

➡️ Most analyst time is spent **cleaning data** before analysis.

---

## 8. ETL
[ETL](ca://s?q=ETL_in_data_analytics) stands for:
- Extract  
- Transform  
- Load  

**Purpose:** Collect data from multiple sources, clean it, and load it into a centralized system for analysis.

---

## 9. Data Visualization
A good [visualization](ca://s?q=Data_visualization_best_practices) should:
- Clearly communicate insights
- Help stakeholders make decisions
- Use the right chart for the right purpose

**Common charts:**
- Bar Chart
- Line Chart
- Scatter Plot
- Histogram
- Pie Chart
- Maps
