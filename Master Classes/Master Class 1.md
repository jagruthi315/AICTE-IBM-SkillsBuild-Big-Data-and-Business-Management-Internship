## 1. What is Data Analytics?
[Data Analytics](ca://s?q=What_is_Data_Analytics) is the process of:
- Collecting
- Cleaning
- Analyzing
- Visualizing  

➡️ Purpose: Extract insights that support better decision-making.

## 6. Quantitative vs Qualitative Data

### [Quantitative Data](ca://s?q=Quantitative_data)
- **Discrete:** Countable values (e.g., Number of employees)  
- **Continuous:** Measurable values (e.g., Height, Weight)

### [Qualitative Data](ca://s?q=Qualitative_data)
- **Nominal:** No order (e.g., Blood Group, Country)  
- **Ordinal:** Ordered categories (e.g., Poor, Good, Excellent)

## 🔑 Key Takeaways
